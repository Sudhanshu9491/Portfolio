## 🚀 How to Run Locally  

1️⃣ Clone the repository:  
```bash
git clone https://github.com/Sudhanshu9491/Portfolio.git
cd Portfolio
```

2️⃣ Install dependencies:
```bash
npm install
```

3️⃣ Start the development server:
```bash
npm run dev
```

4️⃣ Open Browser:
```bash
http://localhost:5173/
```
